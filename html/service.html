Descripción de los Servicios

El sistema de citas médicas ofrece una variedad de servicios diseñados para cubrir las necesidades de nuestros pacientes. Nuestro objetivo es proporcionar atención médica de calidad y accesible.

Tipos de Citas Disponibles

  • Consulta Médica: Citas para consultas médicas generales.
  • Chequeo: Citas para exámenes de rutina y chequeos de salud.
  • Citas de Seguimiento: Citas de seguimiento para monitorear la salud del paciente después de un tratamiento o diagnóstico.

Disponibilidad de Médicos y Especialidades

Nuestro sistema cuenta con un equipo de médicos calificados en una variedad de especialidades médicas, incluyendo:

  • Cardiología
  • Pediatría
  • Ginecología
  • Neurología

Para programar una cita con un médico específico o en una especialidad determinada, Puede utilizar la opcion de agenda de citas.
